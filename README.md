# JSON 工具集

这是一个包含多个 JSON 对比工具的集合，可以在浏览器中直接使用。

## 工具列表

### 1. JSON Deep Compare (`index.html`)
- 深度对比两个 JSON 对象
- 支持忽略数组顺序选项
- 可视化差异展示

### 2. JSON 智能对比 (`json_smart_diff.html`)
- 智能 JSON 对比工具
- 忽略对象键序
- 支持忽略数组元素顺序
- 详细的差异摘要和可视化对比

## 使用方法

直接在浏览器中打开对应的 HTML 文件即可使用。

## 部署

本项目已配置 Vercel 部署：
- 主页：`/` → `index.html`
- JSON 智能对比：`/json-smart-diff` → `json_smart_diff.html`

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无需依赖，可直接在浏览器中运行
